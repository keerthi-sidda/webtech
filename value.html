CSS Value

CSS Values are set against CSS properties and reside within CSS declaration block, which is a part of the CSS rule/statement.Values are used in following properties:
  • used in z-index, line height, counter-increment.
  • used in width, margin, padding, font-size, border-width, text-shadow etc.
  • used in percentages also.
  • used in backgroung image, cursor, list-style etc.
  • used in color,background, border or box shadow.
