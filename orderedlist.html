Ordered List

HTML Ordered List displays elements in numbered format.The HTML ol tag is used for ordered list. We can use ordered list to represent items either in numerical order format or alphabetical order format,or any format where an order is emphasized.There can de different types of numbered list:
  • Numeric Number(1,2,3)
  • Capital Roman Number(I,II,III)
  • Small Roman Number(i,ii,iii)
  • Capital Alphabet(A,B,C)
  • Small Alphabet(a,b,c)
